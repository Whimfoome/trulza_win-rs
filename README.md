# Trulza
This is an external hack made to help new users into memory reading/writing in Rust Language. Everything should be easy to read.

### Includes:
- Memory Injection/Reading/Writing
- Helpers like KeyPress and ThreadSleep
- Features
  - Bhop
  - NoFlash
  - Glow
  - Radar
  - SkinChanger
  - TriggerBot
